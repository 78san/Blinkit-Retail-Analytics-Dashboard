# Blinkit-Retail-Analytics-Dashboard

Project Overview

This project is an interactive Power BI dashboard developed to analyze Blinkit's sales performance, customer ratings, outlet distribution, and product categories. The dashboard provides actionable insights into business operations through dynamic visualizations and KPI tracking.

Key Features
Total Sales, Average Sales, Average Rating, and Number of Items KPIs
Outlet Establishment Year Trend Analysis
Outlet Size Distribution Analysis
Outlet Location Performance Comparison
Item Type-wise Sales Analysis
Fat Content Category Analysis
Outlet Type Performance Dashboard
Interactive Filters for Outlet Location, Outlet Size, and Item Type
Tools & Technologies
Power BI
Microsoft Excel
Power Query
DAX (Data Analysis Expressions)
Business Insights
Analyzed sales performance across different outlet types and locations.
Identified customer rating trends and product category performance.
Compared outlet sizes to evaluate revenue contribution.
Created interactive visuals to support data-driven decision-making.

Conclusion

This dashboard demonstrates how Power BI can transform raw retail data into meaningful business insights. It enables stakeholders to monitor sales performance, understand customer behavior, and make informed business decisions through interactive analytics.
